Weather dashboard 

- A very small Node + Express server that serves a single-page React app.
- The app uses Open-Meteo's API to fetch current weather for a city.

How to run
1. Ensure Node.js is installed 

2. cmd

-cd weather dashboard
-npm install
-npm start

4. Site will run on http://localhost:3000

- No API key required: Open-Meteo is free and public
